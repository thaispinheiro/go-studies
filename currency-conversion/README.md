# Currency Conversion

This project performs currency conversion using Go.

## Structure
- `currency-conversion/` - Main source code
- `go.mod` - Dependency management
- `main.go` - Application entry point

## How to run

1. Install Go (https://golang.org/doc/install)
2. In the terminal, navigate to the `currency-conversion` folder.
3. Run:
   ```sh
   go run main.go
   ```
