# Go Studies

This repository aims to centralize projects and experiments for learning the Go programming language.

## Purpose

- Practice fundamental and advanced Go concepts
- Explore different types of applications and challenges
- Serve as a reference for future studies

## Included Projects

- **currency-conversion/**: Simple currency converter in Go
- **get-one-billion-row-challenge/**: Efficient large data processing challenge

Each folder contains its own README with execution details and specific objectives.

## Learning Roadmap

### Completed
- Go environment setup and installation

- Structs, arrays, slices, and maps
- File handling
- Packages, modules, and project organization
- Goroutines and introduction to concurrency

- Interfaces and polymorphism
- Building simple APIs
